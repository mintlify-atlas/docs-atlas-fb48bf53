## What is Sena Shopping?

Sena Shopping is a marketplace platform built for SENA (Servicio Nacional de Aprendizaje) entrepreneurs to showcase and sell their products. Customers can discover products, save favorites, and contact entrepreneurs — while administrators maintain full oversight of the platform.

<Info>
  This is a React + Vite frontend application. It connects to a backend REST API running at `http://localhost:3000/api`.
</Info>

## Platform roles

The platform supports three distinct user roles, each with its own set of capabilities:

<CardGroup cols={3}>
  <Card title="Customer" icon="user">
    Browse products, search by keyword, filter by category and price, save favorites, and view entrepreneur profiles.
  </Card>
  <Card title="Entrepreneur" icon="store">
    Register as an entrepreneur, publish products with images and pricing, edit or delete listings, and manage your profile.
  </Card>
  <Card title="Administrator" icon="shield">
    Access the admin dashboard, approve or reject entrepreneur accounts, manage all products and users, and generate PDF reports.
  </Card>
</CardGroup>

## Key features

<CardGroup cols={2}>
  <Card title="Product catalog" icon="grid" href="/features/product-catalog">
    Full product listing with category filtering, price range controls, and sort options (newest, lowest price, highest price).
  </Card>
  <Card title="Authentication" icon="lock" href="/features/authentication">
    JWT-based login and registration with automatic session expiry handling and password recovery flow.
  </Card>
  <Card title="Search" icon="magnifying-glass" href="/features/search-and-filters">
    Real-time search across product names, descriptions, categories, and entrepreneur names.
  </Card>
  <Card title="Favorites" icon="heart" href="/features/favorites">
    Save products to a personal favorites list accessible from any device after login.
  </Card>
  <Card title="Entrepreneur panel" icon="pen-to-square" href="/features/entrepreneur-panel">
    Create, edit, and delete product listings with image upload via Cloudinary integration.
  </Card>
  <Card title="Admin panel" icon="gauge" href="/admin/overview">
    Dashboard with platform statistics, user management, product moderation, and PDF report generation.
  </Card>
</CardGroup>
